# 查看进度

## 支持数据类型

| 数据源 | 数据目标 | 全量迁移 | 增量同步 |
|--------|----------|--------|--------|
| MySQL  | MySQL    | 支持     | 支持     |
| MySQL  | TiDB     | 支持     | 支持     |
| TiDB   | MySQL    | 支持     | 不支持   |
| TiDB   | TiDB     | 支持     | 不支持   |
| Redis  | Redis    | 支持     | 支持     |
