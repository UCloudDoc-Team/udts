# 实例类型

UDTS 提供五种实例类型供用户选择。

其中 o.udts.2xlarge/o.udts.4xlarge 配置仅支持以下迁移类型。
- MySQL -> MySQL
- MySQL -> TiDB
- MongoDB -> MongoDB
- Redis -> Redis

## o.udts.4xlarge

不限制磁盘，适合大型传输业务使用。

| 数据类型    |  全量传输速率上限（单位Mbps）| 增量同步 RPS 上限   |
| -------    | -------------| ----------- |
| MySQL      |  1200         |     34000   |
| Redis      |  1200         |    120000   |
| MongoDB    |  1200         |     60000   |

## o.udts.2xlarge

| -------    | -------------| ----------- |

| Redis      |  1600         |    150000   |
| MongoDB    |  1600         |     80000   |

| 数据类型    |  全量传输速率上限（单位Mbps）| 增量同步 RPS 上限   |

| MongoDB    |  80          |     10000   | 
| PostGre    |  80          |     10000   | 


**说明**
线上任务运行时，同步性能受到网络环境、源库和目标库的性能等因素影响，实际性能值可能与上表存在一定差异。
